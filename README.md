从 Gallery 下载 STL 文件。

在打印所有部件之前，您应该打印 CALIBRATOR，以检查您的部件是否适合在一起。如果您很难将这些部分放在一起，调整切片器软件的水平扩展设置可以解决这个问题，此设置可能会因您的切片器和打印机而异，但用户报告将其设置为 -0.15 是一个很好的起点。
